## 关于Utilits Hardware Toolbox

本软件由What_Damon开发，使用了ChenZR的Aurora Hardware Check模块，是一款图吧工具箱，卡硬工具箱和云糕工具箱的轻量替代品。

**注意，本软件尚未发布！**

## Utilits Hardware Toolbox亮点

- 足够简单，没有花里胡哨的功能
- 人性化
- 轻量
